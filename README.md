# 📑 Task manager simulator
This project is a passive loading simulator. I created it before I start to learn databases, so everything I learned in it has been forgotten (except for the MVVM pattern).
## 🛠️ Stack
***
  - WPF
  - MVVM
***
## 📖 What I learned 
- Work with asynchronous methods;
- Repeat MVVM pattern.
## 🚀 How to run it
1. Clone repository in your local machine.
2. Open powershell (or any another command line) and perform the following commands:
```bash
  git clone "https://github.com/T-two-K/temperature-mvvm.git" #(if you didn't complete the previous step)
  winget install Microsoft.DotNet.SDK.10
  cd task-manager-simulator #(or full path to this folder, for example: C:/Users/user123/task-manager-simulator)
  dotnet restore 
  dotnet run --project TaskManagerSimulator.csproj
```
